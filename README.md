# FemboyAI
An AI Femboy Python File That Uses Ollama And Gemma4

# HOW TO INSTALL

## Install [Ollama](https://ollama.com/download)

## Install [Python 2.13.13](https://www.python.org/downloads/release/python-31313/)

## In CMD Type:
## ```pip install customtkinter requests pillow duckduckgo_search```
## ```ollama pull gemma4:e2b```

## Install The [Zip File](https://github.com/GatorMouth/FemboyAI/archive/refs/heads/main.zip)

## Unzip The Folder.

## Double Click The Python File.
