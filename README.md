# FemboyAI
An AI Femboy Python File That Uses Ollama And Gemma4

# HOW TO INSTALL

## Install [https://ollama.com/download](Ollama)

## Install [https://www.python.org/downloads/release/python-31313/](Python 3.13.13)

## In CMD Type:
## 'pip install customtkinter requests pillow duckduckgo_search'
## 'ollama pull gemma4:e2b'

## Install The Zip File [https://github.com/GatorMouth/FemboyAI/archive/refs/heads/main.zip](INSTALL)

## Unzip The Folder.

## Double Click The Python File.
